# 超市管理系统 java593

## 项目概述

本项目是一个超市后台管理体系，是一个线上物品平台，能够进行超市信息管理。系统基于前后端分离的架构，采用Java和SpringBoot作为后端开发技术，Vue作为前端开发技术。

## 技术栈

- 后端：Java、SpringBoot
- 前端：Vue
- 数据库：Mysql

## 系统角色

- 用户
- 管理员

## 运行环境

- 系统环境：Windows，Mac
- 开发环境：IDEA、JDK1.8、Mysql
